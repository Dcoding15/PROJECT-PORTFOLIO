Machine Learning Projects

Category: Machine Learning • Completed: Ongoing

Breast Cancer Prediction - A machine learning pipeline that predicts whether a tumor is malignant or benign with high accuracy. Using data exploration, feature scaling, and Logistic Regression, the model delivers reliable results validated by accuracy scores and confusion matrices. This project showcases the power of AI in healthcare—turning raw data into insights that can support early detection and life-saving decisions.


Housing Price Prediction - This project leverages the Boston Housing dataset to predict house prices based on key socio-economic and structural features. By analyzing factors like crime rates, average rooms, tax levels, and neighborhood demographics, the model uncovers what truly drives property values. The result is a data-driven approach that demonstrates how machine learning transforms raw statistics into actionable real-estate insights.


Loan Approval Prediction – This project leverages a loan dataset of 480 applicants to predict approval outcomes based on key socio-economic and financial features such as income, loan amount, credit history, education, and property area. By analyzing applicant profiles, the model demonstrates how machine learning transforms raw financial data into actionable insights for risk assessment and smarter lending decisions.


Sonar Signal Prediction - This project leverages a renowned sonar dataset of 208 signal patterns to classify sonar returns as either originating from a rock or a metal cylinder (mine). Each pattern is composed of 60 numerical attributes representing the energy within a particular frequency band, captured from bouncing sonar signals off different surfaces.


Stock Prediction - Flask app leveraging Gradient Boosting Regressor and Yahoo Finance API to forecast key stock metrics (open, high, low, close, volume) with 15-minute interval precision. Delivers actionable insights through dynamic charts and next-day predictions for informed trading decisions.


Wine Quality Prediction - A streamlined machine learning pipeline that processes physicochemical wine data. It leverages a Random Forest algorithm on standardized features to perform accurate binary classification. The model's performance is validated through robust training and testing accuracy metrics.


Titanic Survival Prediction - A comprehensive machine learning pipeline leveraging Logistic Regression to forecast passenger survival. Processes demographic and ticket data through meticulous cleaning, one-hot encoding, and feature engineering. Achieves robust 80% accuracy with clear ROC analysis and coefficient interpretation for actionable insights.


Fake News Prediction - A high-performance NLP pipeline leveraging Logistic Regression and TF-IDF vectorization to classify news articles with near-perfect accuracy. Implements advanced text preprocessing including stemming, tokenization, and stopword removal. Achieves 99% precision and recall on test data, demonstrating robust generalization against misinformation.
