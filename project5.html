Mail Form Page: This page allows users to send a message by filling out a simple form. It collects basic information like the sender’s name, email, date, subject, and the message itself.

User Interaction: After entering the details, the user can submit the form using the Submit button, which sends the information to the server for processing.

Confirmation Page: Once the form is submitted successfully, a confirmation page appears with a “Thank You” message to let the user know their mail was received.

Navigation Option: The confirmation page also includes a “Go Back” button so the user can easily return to the previous page if needed.
