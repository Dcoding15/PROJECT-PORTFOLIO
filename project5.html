Mail Form Page

Allows users to send a message by filling out a simple form. Collects sender's name, email, date, subject, and message.

User Interaction

After entering details, users can submit the form using the Submit button, which sends information to the server for processing.

Confirmation Page

Once the form is submitted successfully, a confirmation page appears with a "Thank You" message to confirm mail receipt.

Navigation Option

The confirmation page includes a "Go Back" button so users can easily return to the previous page if needed.

Backend Processing

Django backend handles email sending via SMTP
